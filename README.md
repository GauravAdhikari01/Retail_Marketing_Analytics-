# 🛍️ Retail & Marketing Analytics Project

An end-to-end retail analytics project focused on customer segmentation, sales optimization, KPI tracking, and marketing effectiveness using Python, machine learning, and interactive dashboards. 

---

## 📌 Project Overview

This project analyzes retail transaction data to uncover customer behavior patterns, optimize marketing strategies, and improve business performance through advanced analytics techniques such as:

* RFM Analysis
* K-Means Clustering
* Cohort Analysis
* Market Basket Analysis
* KPI Tracking & Dashboarding

The project is designed to simulate a real-world business analytics case study for retail and e-commerce companies.

---

## 🎯 Business Objectives

* Identify high-value customer segments
* Improve customer retention
* Analyze product and sales performance
* Optimize marketing campaigns
* Build interactive dashboards for stakeholders
* Generate actionable business insights

## 🚀 Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/retail-marketing-analytics.git
cd retail-marketing-analytics
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 3️⃣ Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Dashboard

```bash
python dashboards/dash_app.py
```

Open browser:

```bash
http://127.0.0.1:8050
```
